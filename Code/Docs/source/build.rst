Documentation
-------------
The code for this project attempts to be as clear as possible.
In order to achieve this, both inline commenting and external documentation tools are used.

External documentation for this project is generated using the Sphinx documentation generator.
To generate the documentation for the current release there is a Makefile in the "Doc" folder.
This provides commands to produce documentation in multiple different formats.

For example, to produce the documentation in HTML format, the following command should be run.

.. code::

   $ make html
